# GBC Dashboard

A full-stack dashboard that syncs orders from RetailCRM into Supabase, displays analytics in a Next.js app, and sends Telegram alerts for high-value orders.

---

## 🚀 What I Built

This project implements a complete data pipeline:

1. Import mock orders into RetailCRM  
2. Fetch and sync orders from RetailCRM → Supabase  
3. Display orders and revenue in a Next.js dashboard  
4. Send Telegram alerts for large orders (> 50,000 ₸)  

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), TypeScript  
- **Backend:** Node.js scripts (tsx)  
- **Database:** Supabase (PostgreSQL)  
- **API:** RetailCRM  
- **Notifications:** Telegram Bot API  
- **Deployment:** Vercel
